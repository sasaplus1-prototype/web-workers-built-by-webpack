# web-workers-built-by-webpack

Web Workers built by [webpack](https://webpack.js.org/)

## Setup

install dependencies.

```sh
$ npm install
```

build script.

```sh
$ npm run webpack
```

## Tools

- [webpack](https://webpack.js.org/)
- [worker-loader](https://github.com/webpack-contrib/worker-loader)
